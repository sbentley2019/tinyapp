## Getting Started

1. Clone this repository into a local folder.
2. Install dependencies using `npm install`.
3. Run the app using `npm start`.
4. Visit <http://localhost:8080/>.

## Dependencies

- Node.js
- Express
- EJS
- bcrypt
- body-parser
- cookie-session
